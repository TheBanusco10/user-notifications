# User Notifications

A Wordpress plugin to send notifications to users.

## Usage

Create a notification and choose the role to send it. Also, you can choose between two type of templates: basic and flex.

Use [un_notifications] shortcode to display a list of notifications. Users will be able to see all notifications based on their roles, if an user is not logged in, then he will be redirect to home page. **IMPORTANT** not to add this shortcode on home page.
